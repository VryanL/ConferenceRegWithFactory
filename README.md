# ConferenceRegWithFactory

 ## Features
 * Register people to a conference show a list with all there info.
 * Find by there name.
 * Count how many Attendees there are by Total, Affiliation or Type.
 * Calculate total fees by All or Type.
 * Saves to an XML.
 * Load from an XML.
